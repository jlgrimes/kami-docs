---
title: Kami
description: Native-feeling React UI for Capacitor and mobile web
---

Kami is a React UI kit optimized for mobile UX: haptics, sheets, native-ish gestures, and clean tokens.

## Install

```bash
npm install @jlgrimes/kami
```

## Quickstart

```tsx
import '@jlgrimes/kami/dist/style.css';
import { Button } from '@jlgrimes/kami';
```

Use the sidebar to browse every component page and preview.
